1. About This Tool

Most small nonprofits have committees on paper but not in practice. The committee descriptions are in a long-forgotten governance binder, members aren't sure what they're supposed to do, and the same three people end up doing all the work. This tool fixes that by giving every committee a clear charter, member roster, work plan, and meeting log.

Right-size your committees

An all-volunteer board needs 2-3 standing committees, not 8. The Library is opinionated about which committees actually matter for orgs of different sizes — start there.

2. Getting Started

Demo: demo@example.org / demo — populated with 4 sample committees, sample charters, and a cross-reference matrix.

The onboarding wizard

New teams go through a brief setup that asks about org size and life stage — then recommends the 3-5 committees most appropriate for you. You can accept the recommendations or pick your own from the Library.

3. My Committees & Library

My Committees

Your active committees. Each has a charter, member roster, work plan, meeting calendar, and document library. Click into any committee for the full detail page.

Committee Library

A curated set of best-practice committees with editable charter templates — Executive, Finance, Governance/Nominating, Development/Fundraising, Audit, Programs, Marketing, Strategic Planning, Compliance, Personnel, and more. Add any to your active set in one click.

4. Ad Hoc Committees

Separate workspace for time-bounded task forces — capital campaign committee, search committee, strategic plan task force. Ad hoc committees have a defined sunset date and dissolve when their work is done.

Ad hoc, not permanent

Every ad hoc committee should have a clear end date in its charter. If a task force has been running for more than 18 months, either dissolve it or convert it to a standing committee.

5. Onboarding & Offboarding

When a board member joins or leaves a committee, run them through structured onboarding/offboarding checklists — orientation, expectations, materials, exit interview, file handoff. Documenting transitions consistently is what keeps institutional knowledge from walking out the door.

↑ Back to top

6. Templates & Cross-Reference Matrix

Templates Directory

Pre-built charters, agendas, minutes templates, work plan templates, and orientation packets — organized by committee type. Pull what you need; everything is editable.

Cross-Reference Matrix

A grid showing every governance responsibility (financial oversight, fundraising, programs, compliance, etc.) and which committee owns it. Surfaces overlap and gaps. Use it to right-size committee scope and prevent things from falling through the cracks.

7. Build & Export

Generate a polished annual committee report (HTML + DOCX) summarizing every committee's composition, meeting cadence, key accomplishments, and year-ahead priorities. Drop it into your board packet for the annual meeting.

Administrator Access

The sign-in screen has an Administrator Access link below the Sign In button. Use it to sign in as Administrator with just a password — no email needed. This is a per-browser admin role; the password is stored only on the current computer.

  • First time: Click Administrator Access. You'll see a "First-time setup" prompt with two password fields — enter a password (6+ characters) and confirm it. Click Create Admin Password.
  • Subsequent times: Click Administrator Access, enter that same password, and click Enter Admin Panel.
  • Once signed in as Administrator, you'll land on the dashboard with full admin privileges, including the Admin page in the sidebar (visibility into all teams, users, and activity stored in this browser).
  • Click ← Back to regular sign-in at the bottom of the admin panel to return to the normal email/password form.

Note: the admin password is unique to each browser. If you set it up at home and then visit the app on a work computer, you'll see the first-time-setup prompt again. To grant admin access on a new machine, register a regular user account or set up a fresh admin password there.
